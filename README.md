# 🛠 Calculadora de Energia 

---

## Descrição

Com a calculadora de energia, você poderá se informar do gasto de um eletrodoméstico usado por um período de tempo.

---

## Linguagem

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python&logoColor=yellow)

---

## 🧮 Formula

C=P⋅t⋅V

📌 Onde:
C = custo total (R$)
P = potência do aparelho (kW)
t = tempo de uso (horas)
V = valor do kWh (R$/kWh)

---

## Executar o programa


Se estiver no GitHub:

git clone https://github.com/luisfabiols/consumo-energia
cd consumo-energia

Ou baixe o ZIP e extraia

Execute o arquivo principal:

python app.py
